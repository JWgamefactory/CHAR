# CHAR
CHAR는 대한민국의 비영리 목적 프로그래밍 사이트 '엔트리(playentry.org)'에서
제가 만든 작품의 아이디어를 약간 변형하여 turtle 모듈과 사용할 수 있는 난해한 프로그래밍 언어입니다.
https://playentry.org/project/69ec9ba8daf7a630df2950dd
이 언어는 HQ9+나 Brainfuck처럼 한 줄로 코딩할 수 있는 특징을 가졌으며, 명령의 종류도 수없이 많지만
아이들을 위한 프로그래밍 사이트에서 만든 언어인지라 한계가 많습니다.
각 글자 하나마다 각자 다른 명령을 실행합니다.

|글자|의미|
|:-:|-|
|a부터 z까지의 26가지 알파벳 소문자|현재 사용하는 변수 바꾸기(26가지 지정 가능)|
|A|변숫값(0~127)의 자료형을 정수에서 아스키코드 형태로 전환하기, 아스키코드 상태면 정수로 전환하기 (아스키코드 아니면 10진법 유니코드로 전환)|
|B|짧은 멜로디 재생하기|
|C|펜의 색상값을 1/36만큼 증가시키기|
|D|아래로 10픽셀 이동하기|
|E|turtle 창의 중심으로 펜을 이동시키기|
|F|이동 방향으로 10픽셀 이동하기|
|G|펜을 15도 회전시키기|
|H|1초 딜레이|
|I|만약 변숫값이 10인가? 조건문 실행|
|J|펜 올리기|
|K|변숫값을 1부터 10까지의 무작위 정수로 지정|
|L|왼쪽으로 10픽셀 이동하기|
|M|변숫값 출력하기|
|N|줄바꿈|
|O|그린 것과 펜 데이터 초기화하기|
|P|펜 내리기|
|Q|정수 입력받기|
|R|오른쪽으로 10픽셀 이동하기|
|S|펜의 굵기를 1만큼 증가시키기|
|T|10회 반복문 실행|
|U|위로 10픽셀 이동|
|V|펜 보이기/숨기기 토글|
|W|무한 반복문 실행|
|X|반복문 빠져나가기 (break)|
|Y|반복문 및 조건문의 끝부분|
|Z|프로그램 중단하기|
|0~9까지 숫자|변숫값을 그 숫자로 지정|
|+|변숫값 1 증가|
|-|변숫값 1 감소|
|*|변숫값에 2 곱하기|
|/|변숫값에 2 나누기|
|%|변숫값을 2로 나눈 나머지로 정하기|
|^|변숫값을 제곱하기|
|_|변숫값을 역수로 정하기(1/변숫값)|
|?|펜 모양을 화살표에서 거북이로 바꾸기|
|!|오류 일으키기|
|@|'JW 게임 팩토리' 유튜브 채널 링크 출력 및 홍보하기|
|$|변숫값을 원에서 달러로 환전하기|
|\|변숫값을 달러에서 원으로 환전하기|
|#|사용되는 변수 모두 출력하기|
|&|입력한 코드 출력하기|
|절댓값 기호|주석 (괄호처럼 사용)|

코드 실행시 정의되지 않은 모든 글자는 무시됩니다.
